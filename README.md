# laser-detection-unity

`LaserDetectionUnity` is an iOS protoype with the intention to speed up the testing of laser detection provided by the native plugins
[OpenCViOSPlugin](https://github.com/mantisxsopoci/opencv-ios-plugin). 

<h2> Usage </h2>

1. Open `LaserDetection` in Unity.
2. The required iOS native plugin has been included in the project which can be found in _**Assets > Plugins > iOS**_. 
3. Test the project via mouse in the editor or switch to the iOS platform. 
4. After building to iOS, follow the instructions found in the  README.md of [OpenCViOSPlugin](https://github.com/mantisxsopoci/opencv-ios-plugin).
4. If [OpenCViOSPlugin](https://github.com/mantisxsopoci/opencv-ios-plugin) is updated, remove the old plugin from `LaserDetectionUnity` and follow the instructions found in the native plugin's README.md.
